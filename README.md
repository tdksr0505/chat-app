# Draw & Chat

## 簡介

結合即時聊天室及共同繪畫白板功能的程式。

## demo

demo 影片中開啟兩個視窗模擬多人連線情境<br>
[demo 影片](https://drive.google.com/file/d/1kStyhTCazVG5yc2ncGeMeI1EPu5WITVX/view?usp=sharing)

## 前端 / Server

- Nextjs
- TypeScript
- Tailwind
- Socket.io

## 啟動

### 啟動 client 端

```
yarn dev
```

### 啟動 server 端

```
cd server
yarn server
```
